# LUXDROP — Frontend Application 👟✨

Интерфейс современного интернет-магазина брендовых товаров с китайских площадок (Poizon, Taobao). Проект выполнен в рамках дисциплины **«Проектный практикум»**.

## 🎨 Особенности интерфейса
* [cite_start]**Premium UI:** Дизайн в темных тонах с золотыми акцентами (`#C9A84C`) для соответствия люкс-сегменту[cite: 513, 514].
* [cite_start]**9 функциональных страниц:** Главная, каталог, карточка товара, корзина, избранное, профиль, история заказов, вход и регистрация[cite: 414].
* [cite_start]**Умная фильтрация:** По категориям, брендам и цене с автоматической синхронизацией через URL[cite: 516, 518].
* [cite_start]**State Management:** Использование React Context API для управления корзиной и авторизацией[cite: 511].
* [cite_start]**Адаптивность:** Полная поддержка мобильных устройств и планшетов[cite: 478].

## 🛠 Технологический стек
* [cite_start]**Framework:** React 18 + Vite (ESM)[cite: 495].
* [cite_start]**Styling:** Tailwind CSS (Utility-first подход)[cite: 495].
* [cite_start]**Routing:** React Router v6[cite: 495].
* [cite_start]**API Client:** Axios с настроенными интерцепторами для JWT[cite: 495, 512].
* [cite_start]**Deployment:** Vercel[cite: 369].

## 🚀 Запуск проекта
1. Установите зависимости: `npm install`
2. Создайте файл `.env` и укажите `VITE_API_URL` (адрес вашего бэкенда).
3. Запустите режим разработки: `npm run dev`
