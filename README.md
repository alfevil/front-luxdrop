# LUXDROP — Frontend (React) 👟✨

Интерфейс интернет-магазина брендовых товаров с платформы Poizon. Проект разработан в рамках учебной дисциплины **«Проектный практикум»**.

## 🎨 Особенности интерфейса
* **Premium UI:** Лаконичный темный дизайн с золотыми акцентами (`#C9A84C`) для создания премиального образа.
* **Полноценный функционал:** Реализовано 9 страниц, включая каталог, детальную карточку товара, корзину, список избранного и личный кабинет.
* **Умная фильтрация:** Навигация по категориям и брендам с синхронизацией через URL-параметры.
* **State Management:** Управление состоянием корзины и авторизации через React Context API.
* **Адаптивность:** Сайт корректно отображается на смартфонах, планшетах и ПК.

## 🛠 Технологический стек
* **Framework:** React 18 + Vite.
* **Styling:** Tailwind CSS.
* **Routing:** React Router v6.
* **API Client:** Axios (с поддержкой JWT-авторизации).
* **Deployment:** Vercel.
